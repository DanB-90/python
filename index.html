2. Variables

En Python, una variable es un espacio reservado en la memoria que almacena un valor. A diferencia de otros lenguajes, Python tiene tipado dinámico, lo que significa que no necesitas declarar el tipo de dato explícitamente. Los tipos de datos fundamentales incluyen enteros (int), números con decimales (float), cadenas de texto (str) y valores lógicos (bool). El correcto manejo de estos datos es el primer paso para diseñar algoritmos estructurados.

En Python, una variable es un espacio reservado en la memoria que almacena un valor (como int, float, str, bool). Para hacer programas interactivos, utilizamos la función integrada input().

Importante: Todo lo que el usuario ingresa mediante input("Mensaje: ") es capturado como una cadena de texto (str) por defecto. Si necesitas hacer cálculos matemáticos con ese dato, debes convertirlo usando funciones de transformación (Casting) como int() para enteros o float() para decimales. Por ejemplo: edad = int(input("Ingresa tu edad: ")).

3. Condicionales

Las sentencias condicionales (if, elif, else) permiten que un algoritmo tome decisiones y altere su flujo de ejecución. Funcionan evaluando expresiones que dan como resultado un valor lógico o booleano (True o False). Si la evaluación es verdadera, se ejecuta un bloque de código específico. Son fundamentales para la toma de decisiones algorítmicas y para validar datos de entrada.

Para construir estas expresiones de decisión, utilizamos dos herramientas matemáticas y lógicas fundamentales:

  • Operadores de Comparación: Permiten contrastar dos valores. Incluyen: igualdad (==), diferencia (!=), mayor que (>), menor que (<), mayor o igual que (>=) y menor o igual que (<=). Recordar que un solo igual (=) se usa para asignar variables, mientras que el doble igual (==) se usa para comparar.
  • Operadores Lógicos: Permiten combinar múltiples condiciones en una sola evaluación.
    • and: Retorna True solo si todas las condiciones conectadas son verdaderas.
    • or: Retorna True si al menos una de las condiciones es verdadera.
    • not: Invierte el valor de verdad de la condición (si era True, la convierte en False y viceversa).

Los Diccionarios {} son colecciones no ordenadas que almacenan datos en pares de clave:valor. Son extremadamente útiles para estructurar información compleja, similar a un registro de base de datos o un archivo JSON.

5. Ciclos

Las estructuras repetitivas o bucles son herramientas clave para automatizar tareas. El ciclo for se utiliza para iterar sobre una secuencia conocida (como una lista de datos o un rango numérico determinado). El ciclo while, en cambio, ejecuta un bloque de código indefinidamente mientras una condición específica siga siendo verdadera. Comprender cuándo usar cada uno es esencial para optimizar procesos y evitar "ciclos infinitos".

La función range(): Es esencial cuando usamos for y queremos repetir una acción un número específico de veces. range(inicio, fin, paso) genera una secuencia numérica. Ojo: el número de "fin" no se incluye en la secuencia generada. Por ejemplo, range(1, 5) genera los números 1, 2, 3 y 4.
